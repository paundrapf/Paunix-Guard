# Roadmap

## V0 Core Desktop

- PIN setup.
- Guard state machine.
- Charger and input triggers.
- Alarm system with audio restore.
- Local settings and event history.
- Velopack updater scaffold.

## V1 Desktop MVP

- More complete tray integration.
- Sleep/shutdown blocking polish.
- Better alarm audio device selection.
- Installer packaging pipeline.

## V2 Companion

- Local phone pairing.
- Local web remote.
- Optional webcam snapshot on trigger.

## V3 Cloud Relay

- Cloudflare Pages website.
- Cloudflare Worker and Durable Objects relay.
- Optional cross-network phone alerts.
